# 快速启动指南

## 🚀 5分钟快速上手

### 1. 环境准备
```bash
# 确保已安装 Python 3.8+
python --version

# 克隆项目（如果还没有）
git clone <repository-url>

```

### 2. 安装依赖
```bash
# 创建虚拟环境（推荐）
python -m venv venv

# 激活虚拟环境
# MacOS/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# 安装依赖包
pip install -r requirements.txt
```

### 3. 启动应用
```bash
streamlit run app.py
```

应用将在浏览器中自动打开，默认地址：`http://localhost:8501`

**推荐操作**：

**推荐操作**：

3. 查看"门控机制详解"

### 🌊 扩散模型（新增！）
**学习目标**：理解生成式AI的数学基础

**推荐学习路径**：

#### 第一步：数学理论（15分钟）
1. 阅读前向扩散过程的定义
2. 理解重参数化技巧
3. 查看Beta调度可视化
4. 展开"重参数化公式证明"

#### 第二步：前向扩散（10分钟）
1. 选择"简单图案"或"数字图案"
2. 观察不同时间步的扩散结果
3. 查看数值统计，理解信噪比变化
4. 展开"单步扩散详细过程"

#### 第三步：反向去噪（10分钟）
1. 选择"理想去噪"演示
2. 调整噪声水平，观察恢复效果
3. 切换到"逐步去噪过程"
4. 理解去噪公式

#### 第四步：2D可视化（15分钟）
1. 选择"Swiss Roll"数据分布
2. 观察扩散过程的可视化
3. 查看分布统计
4. 生成扩散动画（可选）
5. 展开"Score函数"，理解梯度场

#### 第五步：图像生成（20分钟）
1. 阅读"DDPM vs DDIM 算法对比"
2. 尝试DDPM采样（标准方法）
3. 尝试DDIM采样（调整步数，观察加速效果）
4. 对比两种方法的结果

**关键参数说明**：
- **扩散步数 (T)**: 50-1000，越大越精细但越慢
- **Beta调度**: 
  - `linear`: 均匀添加噪声
  - `cosine`: 早期保留更多信息
  - `quadratic`: 介于两者之间
- **DDIM步数**: 10-100，推荐20-50步
- **随机性参数 η**: 
  - `0`: 完全确定性采样
  - `1`: 等价于DDPM

## 🐛 常见问题

### Q1: 应用启动失败
**A**: 检查Python版本（需要3.8+）和依赖安装
```bash
python --version
pip list | grep -E "streamlit|torch|numpy|plotly"
```

### Q2: 可视化显示不正常
**A**: 尝试清除浏览器缓存或使用无痕模式

### Q3: 扩散模型运行缓慢
**A**: 
- 减少扩散步数（T）
- 使用DDIM代替DDPM
- 减小图像尺寸

### Q4: 内存不足
**A**: 
- 减少节点数（GNN模块）
- 减少批量大小（实验室模块）
- 关闭其他标签页

### Q5: 想要更大的模型或真实数据
**A**: 本项目专注于教学演示，不适合大规模训练。如需实际应用，请参考：
- Stable Diffusion: https://github.com/Stability-AI/stablediffusion
- PyTorch官方教程: https://pytorch.org/tutorials/
